# UI-Design – Nelly Downloader

## Stil

Modern, klar, dunkel, nicht klobig.

Keine klassische alte Windows-Optik.

Designsprache:

- dunkler Hintergrund
- leichte Kartenflächen
- dezente Akzentfarbe Blau/Violett
- klare Icons
- ruhige Abstände
- gute Lesbarkeit
